About HTML

HTML5 is a markup language used for structuring and presenting content on the world wide web. It is the fifth and latest major version of HTML that is a World Wide Web Consortium (W3C) recommendation. The current specification is know as the HTML Living Standard and is maintained by a consortium of the major brower vendors (Apple, Google, Molilla, and Microsoft), the Web Hypertext Application Technology Working Group (WHATWG).

About CSS3

Cascading Stlye Sheets(CSS) is a style sheet language used for described the presentation of a document written in a markup language like HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript
